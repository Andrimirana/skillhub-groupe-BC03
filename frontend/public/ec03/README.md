SKILLHUB - Projet fil rouge EC01 2026

Présentation:
SkillHub est une plateforme web de formation en ligne développée en HTML, CSS et JavaScript natif.Il y a le côté dedié aux formateurs, et aux apprenants.
Elle permet de découvrir des formations, de filtrer et rechercher des cours, de consulter des témoignages
et de s’inscrire via un formulaire.

STRUCTURE DU PROJET:
index.html → Page d’accueil  
formations.html → Page des formations  
css/styles.css → Styles de la page d'accueil
css/formations.css → Styles de la page formation
js/script.js → Interactions page d’accueil  
js/formations.js → Logique page formations  
assets/images → images et icônes de la page d'accueil
assets/Illustrations → Illustrations de la section hero
assets/Formations → Images contenant les différentes formations de lapage formation

FONTCTIONNALITES PRINCIPALES:

Navigation:
Navigation burfer responsive (mobile)
Liens de navigation accessibles

Modale d'inscription:
Ouverture/fermeture (bouton, overlay, touche Escape)
Piégeage du focus clavier
Désactivation du scroll en arrière plan

Témoignages:
Cartes générées dynamiquement
Navigation par points
Animation d’apparition

Formulaire d'inscription:
Validation des champs (nom, email, mot de passe)
Messages d’erreur et bordures rouges
Message de succès et reset formulaire

Page Formations
Affichage dynamique des cartes
Recherche par nom ou catégorie
Filtres (catégorie, durée, prix)

Responsive & accessibilité:
Design responsive (desktop, tablette, mobile)
Focus visible clavier
Attributs ARIA
Navigation clavier dans la modale

HTML5
